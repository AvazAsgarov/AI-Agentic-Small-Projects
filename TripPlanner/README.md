# Trip Planner AI Agent

**Objective:**  
Generate a 3-day travel itinerary for a selected country using AI.

**Description:**  
This project uses OpenAI's GPT-4o-mini model to:
1. Select a country from a given continent.
2. Generate a detailed 3-day trip plan including activities, timing, and travel tips.

**Tech Stack:**  
- Python  
- OpenAI API  
- python-dotenv  

**Example Output:**

```
Selected country: Japan

Day 1: Visit Tokyo Tower, Shibuya Crossing, and local cuisine.
Day 2: Explore Kyoto temples and gardens.
Day 3: Enjoy Osaka Castle and street food markets.
```
